# Apartments by the River 🌊

🔗 Live Website: https://www.apartmentsbytheriver.bg

A modern, multilingual apartment rental website built with **Next.js** and **Payload CMS**.

This project provides a fast, SEO-friendly frontend combined with a flexible headless CMS for content management.

---

## 🚀 Features

- 🌐 Multi-language support
- ⚡ Server-side rendering with Next.js
- 🧠 Headless CMS with Payload
- 🖼️ Dynamic content management (images, text, listings)
- 📱 Fully responsive design
- 🔍 SEO optimized pages

---

## 🛠️ Tech Stack

- **Frontend:** Next.js, React
- **Backend / CMS:** Payload CMS
- **Styling:** Tailwind
- **Database:** Supabase
- **Deployment:** Vercel

---

## 🧩 Architecture

- Next.js handles the frontend and routing
- Payload CMS manages:
  - Apartment listings
  - Content pages
  - Media uploads
- API connects frontend with CMS
